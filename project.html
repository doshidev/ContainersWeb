Project

To define programming assignment(s) using containers for cloud computing and/or computer science students. In the process of defining the assignments, learn and implement the solution and evaluate its performance. Evaluation entails comparing the similar solution deployed on container and on VM.

    Final description of the assignment

    Outcome: Document describing the programming assignment using containers along with its solution and performance evaluation
